# SkipToAnswer for YouTube 개인정보처리방침

**한국어** | [English](../)

**최종 수정일**: 2026-05-24

SkipToAnswer for YouTube(이하 "확장")는 YouTube 영상 카드의 자극적 제목/썸네일에 대한 답을 자막에서 추출하여 1-2문장으로 보여주는 Chrome 확장 프로그램입니다. 본 방침은 확장이 수집·처리하는 정보의 범위와 목적을 설명합니다.

## 2. 수집·처리하는 정보

확장은 아래 세 가지 범주의 정보만 처리합니다. 그 외의 어떤 정보도 수집·전송하지 않습니다.

### 2-1. 계정 정보 (Google OAuth 로그인 시)

| 항목 | 출처 | 보관 위치 | 용도 |
|---|---|---|---|
| 이메일 주소 | Google OAuth | Supabase, 브라우저 로컬 저장소 | 사용자 식별, 사용량 한도 관리 |
| 사용자 식별자 (UUID) | Supabase 자동 생성 | Supabase, 브라우저 로컬 저장소 | 동일 |
| 액세스/리프레시 토큰 | Supabase Auth | 브라우저 로컬 저장소만 | 인증 유지 |

이름, 프로필 사진, 친구 목록 등 Google 계정의 그 외 정보는 요청·수집하지 않습니다.

### 2-2. 영상 요약을 위한 데이터 (SkipToAnswer 버튼 클릭 시)

사용자가 영상 카드의 SkipToAnswer 버튼을 누른 시점에 한해, 아래 데이터가 운영자 서버(Supabase Edge Function)로 전송됩니다.

- 영상 ID (YouTube의 11자리 식별자)
- 영상 제목
- 영상 자막 (YouTube가 제공하는 공개 자막. 최대 12,000자로 잘림)
- 썸네일 URL

서버는 이 데이터를 Google Gemini API로 전달하여 1-2문장 요약을 생성합니다. **시청 기록, 클릭한 적 없는 영상의 정보, 브라우저 활동, 다른 웹사이트 정보는 수집하지 않습니다.**

### 2-3. 사용량 정보

무료 플랜 일일 한도 관리를 위해, 사용자당 하루 요청 횟수와 카운터 리셋 시각을 Supabase에 기록합니다. 어떤 영상에 대해 요청했는지(사용자별 영상 이력)는 저장하지 않습니다.

## 3. 정보 저장 위치와 보유 기간

| 위치 | 저장 항목 | 보유 기간 |
|---|---|---|
| 브라우저 로컬 저장소 (`chrome.storage.local`) | 토큰, 이메일, 사용자 ID | 로그아웃 또는 확장 제거 시 즉시 삭제 |
| Supabase `profiles` 테이블 | 이메일, 사용자 ID, 일일 사용량 카운터 | 계정 삭제 요청 시까지 |
| Supabase `summaries` 테이블 (캐시) | 영상 ID, 생성된 요약 텍스트 | 무기한 (영상 단위 캐시이며 사용자 정보 포함 안 함) |

`summaries`는 모든 사용자가 공유하는 캐시입니다. 동일 영상에 대한 중복 LLM 호출을 막기 위함이며, 어떤 사용자가 어떤 영상을 요청했는지의 연결은 저장되지 않습니다.

## 4. 제3자 서비스

확장은 아래 외부 서비스를 사용합니다. 각 서비스의 처리 방침이 별도로 적용됩니다.

| 서비스 | 용도 | 처리 방침 |
|---|---|---|
| Google (OAuth) | 로그인 제공자 | https://policies.google.com/privacy |
| Google Gemini API | 자막 요약 생성 (서버에서만 호출) | https://policies.google.com/privacy |
| Supabase | 인증·DB·서버리스 함수 호스팅 | https://supabase.com/privacy |

LLM API 키는 운영자 서버에만 보관되며 확장 클라이언트에는 포함되지 않습니다.

## 5. 광고·분석·추적

확장은 광고, 분석 도구(Google Analytics 등), 제3자 추적기를 사용하지 않습니다. 사용자 행동 데이터를 별도로 수집하지 않습니다.

## 6. 권한 사용 사유

| 권한 | 사유 |
|---|---|
| `storage` | 로그인 세션 토큰과 사용량 정보를 브라우저에 저장 |
| `identity` | Google OAuth 로그인 흐름 처리 |
| `scripting` | YouTube 페이지에 자막 추출 코드를 실행 |
| `webRequest` | YouTube의 자막 트랙 URL을 가로채 자막을 추출 |
| `tabs` | 현재 탭이 YouTube인지 확인 |
| `host_permissions` (youtube.com, ytimg.com, supabase.co) | 위 기능 수행에 필요한 도메인 접근 |
