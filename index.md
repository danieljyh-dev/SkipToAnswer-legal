# SkipToAnswer for YouTube — Privacy Policy

**English** | [한국어](./ko/)

**Last updated**: 2026-05-24

SkipToAnswer for YouTube (hereafter "the Extension") is a Chrome extension that extracts answers from YouTube video subtitles and displays them as 1–2 sentence summaries on top of misleading thumbnails and titles. This document describes what data the Extension collects and processes, and for what purpose.

## 1. Operator

- **Operator**: danieljyh.dev
- **Contact**: danieljyh.dev@gmail.com

## 2. Information Collected and Processed

The Extension processes only the three categories of information listed below. It does not collect or transmit any other information.

### 2-1. Account Information (on Google OAuth sign-in)

| Item | Source | Storage Location | Purpose |
|---|---|---|---|
| Email address | Google OAuth | Supabase, browser local storage | User identification, quota management |
| User identifier (UUID) | Generated by Supabase | Supabase, browser local storage | Same as above |
| Access / refresh tokens | Supabase Auth | Browser local storage only | Session persistence |

Other Google account fields such as name, profile photo, or contact lists are not requested or collected.

### 2-2. Data for Video Summarization (sent only when the SkipToAnswer button is clicked)

Only at the moment the user clicks the SkipToAnswer button on a video card, the following data is transmitted to the operator's server (a Supabase Edge Function):

- Video ID (the 11-character YouTube identifier)
- Video title
- Video subtitles (publicly available captions provided by YouTube; truncated to a maximum of 12,000 characters)
- Thumbnail URL

The server passes this data to the Google Gemini API to generate a 1–2 sentence summary. **Watch history, information about videos that were never clicked, browsing activity, and data from other websites are not collected.**

### 2-3. Usage Information

To enforce the free-plan daily limit, the Extension records the number of daily requests per user and a counter reset timestamp in Supabase. It does **not** store which specific videos a user has requested (no per-user video history).

## 3. Storage Location and Retention

| Location | Stored Items | Retention |
|---|---|---|
| Browser local storage (`chrome.storage.local`) | Tokens, email, user ID | Deleted immediately on logout or extension removal |
| Supabase `profiles` table | Email, user ID, daily quota counter | Until account deletion request |
| Supabase `summaries` table (cache) | Video ID, generated summary text | Indefinite (per-video cache; contains no user information) |

The `summaries` table is a shared cache across all users. Its purpose is to avoid duplicate LLM calls for the same video. No mapping between users and the videos they requested is stored.

## 4. Third-Party Services

The Extension relies on the following external services. Each service's own policy applies independently.

| Service | Purpose | Policy |
|---|---|---|
| Google (OAuth) | Identity provider for sign-in | https://policies.google.com/privacy |
| Google Gemini API | Subtitle summarization (called server-side only) | https://policies.google.com/privacy |
| Supabase | Authentication, database, and serverless function hosting | https://supabase.com/privacy |

The LLM API key is stored only on the operator's server and is not included in the extension client.

## 5. Advertising, Analytics, and Tracking

The Extension does not use advertising, analytics tools (such as Google Analytics), or third-party trackers. It does not separately collect user behavior data.

## 6. Permission Justifications

| Permission | Justification |
|---|---|
| `storage` | Persist the sign-in session token and usage information in the browser |
| `identity` | Handle the Google OAuth sign-in flow |
| `scripting` | Execute subtitle-extraction code on YouTube pages |
| `webRequest` | Intercept YouTube subtitle track URLs to extract captions |
| `tabs` | Determine whether the current tab is YouTube |
| `host_permissions` (youtube.com, ytimg.com, supabase.co) | Access the domains required for the above functionality |

## 7. User Rights

- **Sign out**: Click the extension toolbar icon → Sign out. Local tokens are deleted immediately.
- **Uninstall the extension**: Removing the extension from Chrome's extension management page deletes all locally stored data.
- **Account and server data deletion**: Email danieljyh.dev@gmail.com from your registered address. Your `profiles` record will be deleted within 7 days.

## 8. Minors

The Extension is not directed at users under the age of 14.
